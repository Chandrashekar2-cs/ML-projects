# 📊 ML Projects

A collection of beginner-friendly machine learning projects implemented in Python.  
This repository is designed to help learners explore core ML concepts through practical examples.

---

## 📂 Projects Included
- 🏠 **House Price Prediction** – Predict housing prices using linear regression.
- 🌸 **Iris Flower Classification** – Classify iris species using logistic regression.
- 🚢 **Titanic Survival Prediction** – Predict passenger survival using decision trees.
